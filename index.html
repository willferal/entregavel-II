Entregável II

Este site web tem como função demonstrar alguns algoritmos desenvolvidos em JavaScript.

Número primo

Descrição: Este algoritmo tem como função, avaliar a entrada de um valor e reconhecer se é primo ou não.

Sequência de Fibonacci

Descrição: Este algoritmo tem como função, dar o número da sequencia de Fibonacci equivalente a entrada do usuário.

Contagem

Descrição: Este algoritmo tem como função, Realizar a contagem até um certo número descrito pelo usuário.

Ordenação quicksort

Descrição: Este algoritmo tem como função, Realizar a ordenação de uma sequência de números.

Máximo divisor comum

Descrição: Este algoritmo tem como função, Mostrar o máximo divisor comum de um número escolhido pelo usuário.

Somatório

Descrição: Este algoritmo tem como função, Efetuar o somatório de uma sequencia de números.
